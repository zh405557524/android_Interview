# 2.7 View 基础

## 本章目标

- 建立 View 系统从绘制到自定义控件的基础框架。
- 为后续 View 绘制原理、事件分发和性能优化打底。

## 核心知识点清单

- View 绘制基础
- `measure` / `layout` / `draw`
- 自定义 View
- 自定义 ViewGroup
- `Canvas` / `Paint` / `Path` / `Shader`
- `invalidate` / `requestLayout`
- 屏幕适配
- 多分辨率适配
- 动画基础

## 高频面试题

- View 绘制流程分成哪几个阶段？
- `invalidate` 和 `requestLayout` 有什么区别？
- 自定义 View 和自定义 ViewGroup 的关键点分别是什么？

## 追问方向

- `MeasureSpec` 的三种模式。
- `Canvas` 绘制能力与硬件加速的影响。
- 屏幕适配和 dp、sp、px 的关系。

## 易错点 / 对比点

- `onDraw` 不负责布局。
- `requestLayout` 可能触发重新测量和布局，`invalidate` 主要触发重绘。
- 自定义 ViewGroup 的难点通常在测量和摆放子 View。
