# 3.6 PMS

## 本章目标

- 理解 PMS 在应用安装、解析和组件注册中的职责。
- 能从安装链路视角回答包管理相关问题。

## 核心知识点清单

- 应用安装流程
- 扫描安装包
- 解析 `AndroidManifest`
- 四大组件信息注册
- dex / so 处理基础
- 开机扫描与安装

## 高频面试题

- 应用安装的大致流程是什么？
- PMS 为什么要解析 `AndroidManifest.xml`？
- 安装时 dex 和 so 做了哪些处理？

## 追问方向

- 首次安装、升级安装和开机扫描差异。
- 包解析、签名校验和权限声明。
- 应用信息如何被系统服务查询和使用。

## 易错点 / 对比点

- PMS 不只是“安装器”，它还是包信息中心。
- 清单解析的结果会影响系统后续组件调度。
- 安装流程包含文件、权限、解析、优化多个阶段。
