# 7.8 JNI / NDK

## 本章目标

- 建立 Java 与 Native 层交互的基础认知。
- 能讲清 JNI 调用、so 打包和 Native 崩溃定位。

## 核心知识点清单

- JNI 基础
- Java 与 Native 数据交互
- 动态注册与静态注册
- `CMake`
- so 打包与 ABI
- Native 崩溃基础定位
- JNI 常见坑

## 高频面试题

- JNI 为什么存在，它解决什么问题？
- 动态注册和静态注册各有什么特点？
- Native 崩溃定位通常从哪里入手？

## 追问方向

- `JNIEnv`、引用类型和线程附着。
- ABI、so 加载和构建配置。
- 内存管理、字符串转换和异常处理。

## 易错点 / 对比点

- JNI 交互成本不低，不适合滥用。
- 动态注册更灵活，静态注册更直观。
- Native 问题常涉及线程、内存和生命周期边界。
