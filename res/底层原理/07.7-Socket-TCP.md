# 7.7 Socket / TCP

## 本章目标

- 建立 TCP 长连接和 Socket 通信的基础认知。
- 为弱网、心跳、重连和自定义协议设计打底。

## 核心知识点清单

- TCP 三次握手基础
- 长连接设计
- 心跳机制
- 粘包拆包
- 重连机制
- 自定义协议基础
- NDK Socket 通信

## 高频面试题

- TCP 三次握手的目的是什么？
- 长连接为什么需要心跳和重连机制？
- 粘包拆包问题通常怎么处理？

## 追问方向

- 心跳间隔、超时判定和网络切换。
- 包头包体、自定义协议和消息边界。
- Java Socket 与 NDK Socket 使用差异。

## 易错点 / 对比点

- TCP 是字节流，没有天然消息边界。
- 长连接设计重点是稳定性，不只是保持在线。
- 弱网场景下重连要考虑退避和状态恢复。
