# 2.11 网络与序列化

## 本章目标

- 建立 Android 常见网络基础与数据编码方式的认知。
- 为后续网络框架、长连接和多媒体链路做铺垫。

## 核心知识点清单

- Http / Https 基础
- TCP / UDP 基础
- JSON 解析
- 序列化与反序列化
- 长连接与短连接基础

## 高频面试题

- HTTP 和 HTTPS 的核心差异是什么？
- TCP 和 UDP 分别适合什么场景？
- 序列化方案在性能、体积和兼容性上怎么权衡？

## 追问方向

- TLS 握手、证书校验和中间人攻击。
- 长连接心跳、断线重连和连接复用。
- JSON、Parcelable、Protobuf 的场景差异。

## 易错点 / 对比点

- HTTPS 不只是在 HTTP 外面“加密一下”。
- TCP 可靠但不代表实时性一定最好。
- 序列化选择要看通信边界和协议约束。
