# 1.1 Java 语言基础

## 本章目标

- 建立 Java 基础语法、对象模型与常见语言机制的统一认知。
- 能把语言特性讲成面试中的“定义 + 场景 + 对比 + 常见坑”。

## 核心知识点清单

- 面向对象：封装、继承、多态
- 抽象类与接口
- 重载与重写
- 值传递与引用传递
- `==` 与 `equals`
- `hashCode` 与 `equals` 契约
- `String`、`StringBuilder`、`StringBuffer`
- `final`、`static`、`this`、`super`
- 异常体系：checked / unchecked
- 泛型：泛型擦除、上界下界
- 反射
- 注解
- 内部类、匿名内部类、静态内部类

## 高频面试题

- 抽象类和接口的核心区别是什么，分别适合什么场景？
- `==`、`equals`、`hashCode` 三者在集合中如何联动？
- Java 只有值传递这句话该怎么解释？

## 追问方向

- `String` 为什么不可变，以及常量池带来的影响。
- 泛型擦除对运行时类型判断和 JSON 解析的影响。
- 反射与注解分别适合编译期还是运行期能力扩展。

## 易错点 / 对比点

- “引用传递”是常见误区，Java 传递的是引用副本。
- 重载是编译期多态，重写是运行期多态。
- `StringBuilder` 偏单线程性能，`StringBuffer` 偏线程安全。
