# 6.2 Flutter 基础

## 本章目标

- 建立 Flutter 框架核心对象和 UI 构建方式的认知。
- 能讲清 `Widget`、`Element`、`RenderObject` 三棵树关系。

## 核心知识点清单

- `Widget` / `Element` / `RenderObject`
- `StatelessWidget` / `StatefulWidget`
- `setState` 原理
- `BuildContext`
- 生命周期
- 路由管理
- 状态管理基础

## 高频面试题

- Flutter 的三棵树分别负责什么？
- `StatelessWidget` 和 `StatefulWidget` 的核心差异是什么？
- `setState` 为什么能触发界面刷新？

## 追问方向

- `BuildContext` 的作用和常见误用。
- Widget 重建与 Element 复用。
- Flutter 路由与页面管理方式。

## 易错点 / 对比点

- Widget 更像配置描述，不是最终渲染对象。
- `setState` 触发的是标记重建，不是立即同步绘制。
- 生命周期要结合 `State` 对象来讲。
