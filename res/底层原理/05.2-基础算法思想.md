# 5.2 基础算法思想

## 本章目标

- 建立算法题常见解题范式的整体认知。
- 能把复杂度、思路演进和常用套路讲清楚。

## 核心知识点清单

- 时间复杂度
- 空间复杂度
- 递归
- 回溯
- 双指针
- 滑动窗口
- 二分查找
- 排序
- 贪心
- 动态规划
- BFS / DFS

## 高频面试题

- 时间复杂度和空间复杂度应该怎么分析？
- 双指针、滑动窗口、二分查找分别适合哪些题？
- 动态规划题通常怎么找到状态和转移？

## 追问方向

- 从暴力解到优化解的推导过程。
- 递归与迭代、DFS 与 BFS 的选择。
- 贪心成立条件和 DP 对比。

## 易错点 / 对比点

- 复杂度分析不要漏掉隐藏循环和递归深度。
- 回溯和 DFS 相关但不完全等同。
- 二分查找最容易出错的是边界和模板。
