# 1.5 Kotlin 基础

## 本章目标

- 建立 Kotlin 语法糖背后的编译与设计思维认知。
- 能把 Kotlin 和 Java 的差异讲成面试中的加分项。

## 核心知识点清单

- `val` / `var`
- 空安全
- `data class`
- `object` / `companion object`
- 扩展函数
- 高阶函数
- lambda
- `inline` / `noinline` / `crossinline`
- 密封类 `sealed class`
- 委托 `by` / `lazy`
- 集合操作符
- 协程基础语法

## 高频面试题

- Kotlin 空安全是如何在语法层面降低 NPE 风险的？
- `inline` 的作用是什么，为什么会引出 `noinline` 和 `crossinline`？
- `data class`、`object`、`sealed class` 分别适合什么场景？

## 追问方向

- 扩展函数的静态分发特性。
- 属性委托和懒加载的实现思路。
- 协程语法和线程的关系。

## 易错点 / 对比点

- `val` 不等于绝对不可变，只是引用不可重新赋值。
- 扩展函数不会真正修改原类。
- 协程不是线程，也不是自动更高性能。
